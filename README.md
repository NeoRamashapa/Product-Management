# Spring Boot CRUD API

A simple RESTful CRUD application built using Spring Boot to practice backend development concepts.

## Tech Stack
- Java
- Spring Boot
- Maven
- H2 / MySQL
- JPA / Hibernate

## Features
- Create a product
- Read product
- Update product
- Delete product
- REST API endpoints

## How to Run

1. Clone the repository
2. Run:

mvn spring-boot:run

3. Access API at:
http://localhost:8089
